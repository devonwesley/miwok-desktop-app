### Prerequisites

- NodeJS
- Docker
- Yarn

### Installing Miwok

`yarn build`

### Starting Miwok

`yarn start`

### Testing 

Folders: 

```bash
.
└── helpers
    └── index.js
```

Commands:

```javascript
yarn run test
```
